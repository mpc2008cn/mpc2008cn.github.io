Vultr的vps搭建shadowsocks翻墙

稻草人 10月 22, 2015
  • 教程
  • 翻墙
  • 分享到 Twitter

之前花了400大洋买了3年的vpn,之后各种连不上,GFW各种打压vpn,为之苦恼.有天,偶然发现同事用的翻墙工具很快,问之,说是自己的vps搭的shadowsocks.当时就感兴趣了,果断办了个招行的visa卡,开始了折腾之旅.

购买vps

到vultr的官网https://www.vultr.com/freetrial/ 注册账户
目前注册的新用户是给你$50,用2个月,很给力吧~
如何注册不细说,需要注意的是你得有支持外币支付的信用卡,并且这个优惠活动不支持paypal

vps部署

服务器选择

选择东京或者美国西海岸的服务器,速度相对比较快,我选的日本跟洛杉矶的全都卖光了:)

操作系统

主要参考支持锐速的服务器列表http://dl.serverspeeder.com/ls.do?m=availables

我选的Debian 7 x64,之前选的centos跟ubuntu都不能完美开启锐速Debian 8 x64

选择硬件

一个月可以免费用$25的配置,那就$16,$8各一台吧(现在只能选$20+$5了)

下载securecrt并等待系统安装完成

在等待系统安装的时候下载securecrt:https://drive.google.com/open?id=0ByuNnrUBU_n3ZkRMZGhpeXlFYVU

安装好了,是Stopped,没事,点击Manage

选择Restart,等待重启成功
复制服务器的IP地址,初始用户名,初始密码

securecrt连接vps

解压下载的secure crt,双击\SecureCRT_EN\SecureCRT\中的SecureCRT.exe

输入IP地址,用户名,点击connect

弹出框点击Accept&Save

输入密码,勾选save password,OK

出现如图所示,标签变绿就说明连接成功啦.

部署shadowsocks

用的一键安装脚本,参考https://teddysun.com/342.html

wget --no-check-certificate https://raw.githubusercontent.com/teddysun/shadowsocks_install/master/shadowsocks.sh
chmod +x shadowsocks.sh
./shadowsocks.sh 2>&1 | tee shadowsocks.log

共三行,依次复制到securecrt右键粘贴执行
其中的第三个执行后第一步要输入密码(shadowsocks的密码):

第二步输入shadowsocks的端口,我喜欢用888:)

看到这一句时按任意键(比如回车)

等待安装结束,出现下图就表示安装成功啦~

记住server ip,server port,password

开启锐速(已停止注册)

摘自http://blog.deartanker.com/post/3659.html
首先注册一个锐速账号:http://my.serverspeeder.com/
然后在securecrt里依次执行:

wget http://my.serverspeeder.com/d/ls/serverSpeederInstaller.tar.gz
tar xzvf serverSpeederInstaller.tar.gz
bash serverSpeederInstaller.sh

到email address要手动输入了

注册的锐速用户名邮箱,密码
在图中框起来的部分最后2步输入y,别的直接回车

修改配置

securecrt里输入

nano /serverspeeder/etc/config

rsc,advinacc,maxmode均修改为1(如果是Digital Ocean的vps,gso也要改为1)
按ctrl+X组合键退出,再输入y保存修改
再按回车写入配置文件

最后输入下列代码重启锐速即可

/serverspeeder/bin/serverSpeeder.sh restart

开启TFO(TCP Fast Open)

主要参考http://www.shadowsocks.co.uk/shadowsocks-tcp-fast-open/
securecrt输入以下命令:

nano /etc/rc.local

在exit 0之前输入并保存

echo 3 > /proc/sys/net/ipv4/tcp_fastopen

执行

nano /etc/sysctl.conf

输入并保存

net.ipv4.tcp_fastopen = 3

打开ss配置文件

nano /etc/shadowsocks.json

修改fast_open的值为true,保存(不包括逗号)

重启shadowsocks

/etc/init.d/shadowsocks restart

加密方式改为chacha20(不想折腾,愿意用默认加密的可跳过此步)

chacha20安全性接近AES,还是谷歌研制的算法,在移动端使用还可以省电,就折腾了一下
主要参考http://xionghaizi.blogspot.com/2015/05/shadowsocks-pythonchacha20.html

  1. securecrt输入以下命令:
    wget https://download.libsodium.org/libsodium/releases/LATEST.tar.gz
tar zxvf LATEST.tar.gz
cd libsodium-1.0.8
./configure
make && make check
make install
  2. 之后将shadowsocks的配置里加密方式改为chacha20,并保存
    nano /etc/shadowsocks.json
  3. 修复关联
    echo /usr/local/lib > /etc/ld.so.conf.d/usr_local_lib.conf
ldconfig
  4. 重启shadowsocks,服务器端就好了
    /etc/init.d/shadowsocks restart
  5. 修改各个客户端的加密方式为chacha20

windows端运行shadowsocks客户端

到https://github.com/shadowsocks/shadowsocks-windows/releases 下载最新的Shadowsocks-*.zip
解压后运行Shadowsocks.exe

服务器IP,服务器端口,密码对应之前的server ip,server port,password,确定

右键shadowsocks快捷图标,选择启用系统代理
现在点一下谷歌 youtube 是不是都可以了:)

pc端chrome配合SwitchyOmega扩展快速切换

https://chrome.google.com/webstore/detail/proxy-switchyomega/padekgcemlokbadohgkifijomclgjgif?utm_source=chrome-app-launcher-info-dialog
装好后直接导入我的备份即可(从备份文件恢复),省去频繁改动pac.txt
备份文件路径:https://drive.google.com/open?id=0ByuNnrUBU_n3TnRoZUhycGV2VkU

PC端双边TCP加速FinalSpeed,谁用谁知道:)

http://www.d1sm.net/forum-44-1.html

shadowsocks安卓客户端

https://github.com/shadowsocks/shadowsocks-android/releases
具体用法就不细说了,跟windows端配置大同小异

shadowsocks ios客户端

shadowsocks app只是个浏览器,但是ios9出了个神器surge
教程参考 http://www.jianshu.com/p/de1eb844915d
配置参考 https://github.com/liuchenx/SurgeConfig

注意事项

  1. pc上的shadowsocks只适合浏览器翻墙,要想非浏览器翻墙还得再装一个Proxifier,具体可以谷歌,我还没试出效果,所以就不贴了.
  2. 只要发现某个网站打不开,怀疑被墙了,就到shadowsocks.exe同目录下的pac.txt里,按现有的格式添加即可,很方便
  3. 安卓端可以全局翻墙,也可以指定app翻墙,具体自己摸索
  4. 不用试我截图里的vps了,我已经销毁了:)
  5. surge可以某宝买账号:)
